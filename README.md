# bemindtest

## Installation API

* Fork this repository.
* `$ git clone git@github.com:<your username>/bemind-test.git`
* `$ cd api/`
* `$ pip install -r requirements.txt`
* `$ python manage.py migrate`
* `$ python manage.py runserver`

## Installation Web application

* `$ cd web/`
* `$ npm install -g bower`
* `$ npm install`
* `$ bower install`
* `$ grunt serve`
